<block element 블록 요소> 
div / h1~h6 / p / ul / li ....
1. 한줄씩 차지한다(다음에 사용하는 태그는 줄바꿈) width 값을 지정해도 한줄 차지하는건 동일함
2. width 속성과 height 속성이 적용 가능하다.
3. 바깥(외부)여백 margin   안쪽(내부)여백 padding  4방향 지정 가능
4. width 속성을 지정하지 않으면 감싸고 있는 부모 태그의 width 크기를 따라서 100% 값을 가지고 나타난다.
5. height 속성을 지정하지 않으면 해당태그 안에 있는 자식태그들 만큼 높이값이 자동으로 늘어난다.(기본값은 height:auto;)   
   height 속성은 필요에 따라 상황에 따라 작업자가 판단!


<inline element 인라인 요소>
a / span ......
1. 같은 인라인 요소를 가지고 있는 태그들끼리는 옆으로 나열된다.
2. width 속성과 height 속성은 지정 불가
3. margin 여백과 padding  위, 아래 여백 지정이 불가능하다.  왼쪽,오른쪽만 여백 지정 가능
4. 인라인 요소의 기본 가로,세로 크기는 해당태그안에 있는 내용만큼만 잡힘 (width:auto; height:auto;)


<inline-block 인라인블록 요소>
img / input / button ......
1. 같은 인라인 요소를 가지고 있는 태그들끼리는 옆으로 나열된다.
2. width 속성과 height 속성은 지정 가능
3. margin 여백과 padding 여백은 4방향 지정 가능
4. 인라인블록요소에 해당하는 태그(이미지태그)는 해당 이미지 사이즈 만큼만 표시된다. 




태그가 가질수 있는 사각형 모양의 사이즈 개념
